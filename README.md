# Powells_minimization_travis

Powell's quadratic interpolation minimization method

Returns localization of local minimum

Powell_minimize(f, a, h = 0.5, eps = 1e-8, nmax = 100)

f - function to be minimized
a - start point
h - initial step size
eps - accuracy
nmax - maximum steps
